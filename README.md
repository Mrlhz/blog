# docsify个人博客

- [docsify](https://docsify.js.org/#/zh-cn/)


### 快速开始

```bash
npm i docsify-cli -g
docsify init ./docs
```

### 本地预览网站

```bash
docsify serve docs
# or 端口号默认3000
docsify serve docs -p 3000
```

浏览器打开 `http://localhost:3000` 即可。
